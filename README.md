# Setup
```bash
$ composer require tsjost/Tradera
```

# Usage
```php
<?php
$C = new tsjost\Tradera\Category(20);
foreach ($C->get_items() as $item) {
	echo "\"{$item->shortDescription}\" available for {$item->price} SEK until {$item->endDate}\n";
}
```
